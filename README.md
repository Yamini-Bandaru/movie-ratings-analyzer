# Movie Ratings Analyzer using Hadoop MapReduce

This project is a web application that simulates Hadoop MapReduce processing to analyze movie ratings data.

## Features
- Input movie ratings data
- Simulated MapReduce processing
- Calculate average ratings
- Identify top-rated movie
- Interactive and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Enter movie data in format: MovieName,Rating
2. Click "Analyze Data"
3. View results instantly
